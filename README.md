# flatten_the_queue

A project for WirVsVirus

## Installation

- Clone this repo
- `$ npm install`
- `$ npm start`
- `$ cd server && cp .env.example .env # then edit .env`

## Process
- We use the project board
- We create PullRequests
- Someone else approves the PullRequest
